# WowFit CV Prototype — Desktop v2

Обновлённая компьютерная версия тестового приложения.

## Добавлено

- три упражнения: приседания, отжимания, планка;
- два варианта отжиманий:
  - классические;
  - от колен;
- индивидуальный лидерборд;
- командный лидерборд;
- сохранение результатов в `localStorage`;
- система очков:
  - приседание — 1 очко;
  - отжимание — 2 очка;
  - планка — 1 очко за секунду.

## Как работает режим «От колен»

Для классических отжиманий проверяется линия:

```text
плечо → таз → стопа
```

Для отжиманий от колен проверяется линия:

```text
плечо → таз → колено
```

В обоих режимах повтор считается по углу локтя:

```text
руки выпрямлены → нижняя фаза → руки выпрямлены
```

## Запуск

```bash
python -m http.server 8765
```

Открыть:

```text
http://localhost:8765
```

## Важно

- приложение рассчитано на компьютерную версию;
- камера должна стоять сбоку;
- большая часть тела должна быть видна;
- MediaPipe и модель загружаются через интернет;
- видео обрабатывается локально в браузере;
- лидерборд прототипа не использует backend и хранится только в этом браузере.
